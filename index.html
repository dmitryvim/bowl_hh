# Тестирование Было интересно разобраться с набором технологий для языка с которым общаюсь по работе, поэтому выбор технологий такой: - Язык: C - Сборка: Makefile. - Тестирование: Минимальный хэдер на 20 строк кода. - Code coverage: gcc для сбора статистики + lcov для html отчета. [Отчет о покрытии кода.](./coverage_html/index.html) JUnit и рефакторинг, доступный в IDEA, изучаю в рамках другой домашки. # Практики CI беслатно и облачно - TravisCI. Интегрируется с GitHub, собирает автоматически при пуше. Настраивается через файл ".travis.yml" в корне проекта. В артефактах сборки - лог тестов и html с покрытием тестов. Артефакты загружаются в Amazon S3. Ревью - пока нет.
